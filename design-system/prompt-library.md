# VitaCore Image Generation Prompt Library

Apply this shared suffix to every prompt:

> Original fictional healthcare brand, bright clinical confidence, white and icy-blue palette, cobalt blue and surgical green accents, realistic materials, soft daylight, natural skin tones, generous negative space, premium editorial art direction, no visible patient data, no text, no logos, medically plausible, no purple, no horror, no uncanny anatomy.

Use `image-strategy.md` before selecting a prompt. Not every section should contain an image, and only low-density narrative sections should use atmospheric backgrounds.

## Prompt safeguards

- Add exact region, age range, and accessibility representation intentionally.
- Ask for medically accurate equipment by name only when reviewed by a subject-matter expert.
- Never generate patient charts, diagnostic claims, or real credentials.
- Treat generated clinician and patient images as illustrative; disclose AI generation where required.
